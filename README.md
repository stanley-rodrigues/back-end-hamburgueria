# back-end-hamburgueria

###Sobre

CRUD- rotas de GET, POST, PUT, PATCH e DELETE, utilzando também express, cors e UUID v4.

###Inicializar
Faça um clone do repositório, abra no vscode. No terminal digite o comando "npm i" após "npm run dev"
